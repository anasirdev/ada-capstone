Let's get started!

If you haven't already done so, you will need to get a .ged file to upload. Here's how to do that:

  1. Log in to your favorite genealogy site where you already have a family tree built.
  2. make expandable for top three genealogy site instructionsFor example, on ancestry.com, while viewing your tree click the drop-down and choose tree settings. On the right, near the bottom, click on DOWNLOAD YOUR GEDCOM FILE. Save the file to your computer.
  3. Click Choose File and select the .ged file you have downloaded.
  4. Click Next!

this should trigger the creation of a new tree with automatically assigned user.
